Project Design


Project Name: FAAST Company Website


Project Overview:

  • The purpose is to create an interactive website for the FAAST towing company. The site will help customers gain information on the company as well as access information regarding the services that are provided.
  • Intended users will be current and prospective customers in need of roadside assistance.
  • The website will display basic information about the company itself along with pages for each service offered by the company.

Dynamic Functionality:

The website will have multiple types of dynamic functionality including forms, calculators, and data input. The pricing/quotes page will have multiple quote calculators for each service that the company provides. Customers will be able to receive custom quotes based on the service they need and the specifics of their situation. The testimonials page will allow customers to write a testimonial and add it to the site. There will be a list of previous testimonials (hard coded into the site for the sake of this project) and there will be a form with a textbox and a submit button to allow users to add their own testimonials. The added testimonials will not be saved due to the lack of a database. However, the basic functionality will still be present and can be incorporated with a database if the client wanted.
